# Hostel Leave Management System

A modern multi-page web application designed to digitize hostel leave requests and approvals.  
The system improves hostel security and student safety using QR-based outpass verification and parent notification features.

## Features

### Student Module
- Secure student login and registration
- Apply for hostel leave
- View leave status
- Leave history tracking
- QR-based outpass generation

### Admin/Warden Module
- Dashboard for leave requests
- Approve or reject applications
- Student leave monitoring
- Entry and exit verification
- QR code scanner support

### Parent Notification
- Email/SMS notifications (planned)
- Real-time leave updates

### UI/UX
- Responsive multi-page interface
- Clean and user-friendly design
- Mobile-friendly layout
