Hostel Leave Management System

A multi-page web application designed to digitize hostel leave requests and approvals. The system improves hostel security and student safety using QR-based outpass verification and parent notification features.

🚀 Features
Student Login System
Leave Application Portal
Warden Approval Dashboard
QR Code-Based Outpass Generation
Parent Notification System
Leave Status Tracking
Secure Entry/Exit Verification
Multi-Page Responsive UI
